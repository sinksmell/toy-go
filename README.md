# toy-go
Some useful tools for gopher

- [x] Json to Go
- [ ] DDL to Go
- [ ] To be continue ...

## setup

1. Run in Docker

```shell
docker run -p 9090:9090 destory/toy-go:v0.0.1
```

2. Visit http://localhost:9090

## usage

### Json to Go

> A tool that can quickly convert json to go code.

1. Input Json 
2. Click The Icon
3. Copy Go Code
